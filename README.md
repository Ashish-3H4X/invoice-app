## About the Project

This **Invoice Management System** is a web-based application developed using **Laravel** as part of an **internship / academic project**.

The main objective of this project is to provide a simple and efficient way to manage customers, generate invoices, and track payments for small businesses or service-based organizations.

The application follows Laravelâ€™s MVC architecture and standard development practices.

---

## Features

- User authentication (Login / Logout)
- Customer management (Add, Edit, Delete)
- Invoice creation and management
- Automatic invoice number generation
- Invoice status tracking (Paid / Unpaid)
- PDF invoice generation
- Secure data handling
- Clean and responsive user interface

---

## Technology Stack

- **Backend:** Laravel (PHP)
- **Frontend:** Blade Templates, HTML, CSS, JavaScript
- **Database:** MySQL
- **ORM:** Eloquent
- **Authentication:** Laravel Auth
- **PDF Generation:** DomPDF

---

## Project Structure

- MVC architecture
- RESTful routing
- Eloquent ORM for database operations
- Middleware for route protection
- Form validation and error handling
